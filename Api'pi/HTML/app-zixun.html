            • 我忘记密码了要怎么操作？

              忘记密码时请点击登录框下面的忘记密码，输入您注册时的邮箱，登录邮箱按提示输入新密码，登录时使用新密码登录即可。

            • 我注册时带*的都要填写吗？

              出现这种问题一般为以下几种情况，供您参考：注册时带*的都需要您填写，为了更好的为您服务，请填写真实信息，后期如果有问题需要我们解答或者远程协助时能够更方便的联系到您。

            • 如何开始做手机应用？

              点击页面右上方的“开始制作”按钮，进入模式选择界面选择您需要的模式，点击“立即制作”便可进入配置端开始创作您的专属应用了。

            • 制作好的应用在哪里查看和下载？

              点击页面右上角”我的应用”，进入即可查看您已制作生成和未生成的所有应用。点击每个应用右侧的“下载”按钮可选择下载该应用的Android/iPhone版本，或点击旁边的二维码，用手机扫描便可自动检测系统下载安装您的应用。

            • 爱皮皮应用生成的手机应用支持哪些手机？

              目前应用生成的手机应用适用于Android系统(2.3及以上)和IOS系统（4.3及以上）。

            • 如何将制作好的应用上架？上架有什么用？

              每个应用栏的右下方都有“申请上架”按钮，点击后填写应用上架信息，点击“提交申请”发起申请，平台将在1-5个工作日内进行审核，通过后该应用即上架在应用广场中，被所有用户欣赏和分享。菜鸟平台有专门的推广人员，网站用户数量逐步增多，上架后您制作的应用可以更快的被人下载使用。

            • 制作应用时，占位应该怎么设置？

              1.内容一屏幕可展示完时只需要一个占位即可；2.内容过多一屏展示不完时，需要上下滑动展示下面的内容，这时需要出现滚动条，需要做2个占位。第一个占位不需要点自适应，只需把宽度高度设置成你需要的大小，需要点竖向滑动。第二个占位需要点自适应，这时不需要点。注意：刚开始做没有内容时，高度可先设置为100%，等内容填充完，把第二个占位点下自适应即可，如若没有内容就点自适应，则会出现手机上展示没有内容显示，因为第二个占位这时的高度为0，出现此种情况只需把高度修改下即可。

            • 我制作了按钮时，为什么手机显示点击不到？

              不管是头部导航、底部菜单又或是文本内容，做按钮时一般先做个占位，宽度高度设置成自己需要的大小，选择按钮或者图片控件，选取需要的按钮背景。这样做的好处是：假如直接选取按钮或者图片控件做的话，这时的大小设置比较局限，仅仅是按钮的实际大小，在外面套个占位是为了使用户在手机点击时更加方便，占位宽度可用%设置，这样点击面大，点击起来灵活有效。

            • 我制作了多行文本时，内容过多文本框为什么不能全部把文字包起来？

              文字内容过多时需要用到多行文本控件，把文字内容写到文本内容输入框里，这时会出现文字内容跑到黄色区域线的外面了，这时需要点击自适应，我们看到效果还是没有任何变化，还需要点击最上面的设计预览，再转到设计，这时就有效果了。

            • 图片变形该怎么调整？

              做图片满屏展示时，为防止图片变形，可只设置宽度为100%，高度点自适应即可。如果页面内容多，图片不是满屏显示，图片底部使用了占位控件时，比如一张图大小200px*200px，这时候给图片定义宽度高度200px，在它下面做个占位填充个颜色，图片需要处理下左右拉下渐变，让它跟背景颜色融合下，这样做出来的图片是不变形的。如果没有使用占位，只是用横向分割划分版面内容，这时图片高度可选择自适应。

            • 我设置的浏览器为什么外面有条线框？

              浏览器控件默认是有边框的，如果不需要把边框颜色选择无，宽度高度成设置需要的大小。

            • 我设置的列表控件制作了好几行，设置的圆角为什么显示有问题？

              不管是图文列表还是文字列表最好在外面套个占位，简单说就是需要点击的部分都需要套个占位在最外面，这样宽度可用%设置，也可以设置具体数字，看内容需要，这样做是方便手机点击，点击面大使用方便。列表控件使用多样，可在图文列表、文字列表使用，高度需要自己设置具体数字。有一点需要注意的是，列表增加是从上到下增加的，意思是刚开始你规划的是3条内容，后又需要增加1条，这时你会发现新增加的内容在最上面，加入你做的内容多的话，最好刚开始就设置好。如果是没办法设置的话这时最外面的占位就体现出作用来了，你可以把第二个1的内容找到最外面的占位复制或剪切，把第一个1的内容删掉，把刚复制或剪切的内容复制到第一行里面，以此类推，最后一个就可以添加你新增加的内容了。还需要注意，列表展示需要提前规划好行数量，如果加了圆角半径，这时你设定的行数量是3，，这个效果是没有问题的。假如这时候你把3行修改为5行，你可以看到行与行之间的线不是直线而变成圆角半径为10，这样的效果让你不太满意，所以做列表时，可看图，先把行数预设好在开始设置内容就没有问题了。

            • 我设置的横向分割在大屏幕手机上下面内容要怎么全部显示？

              内容过多时使用横向分割可以使屏幕栏目规划更加清晰。因为手机屏幕高度不同，图文列表展示内容多，图文列表最外面的横向分割在这时需要点余下高度，点击余下高度以后手机屏幕不论高度是多少都可以滚动显示完整。

            • 我设置的响应怎么在手机上显示错误？

              做点击响应时需要注意：最好是选中占位做响应，如按钮，只选择按钮图片的话容易出现用户使用起来点击了没反应，这是因为图片的大小是固定的，占位可以设置大一些，使用起来更加方便灵活。还有一点需要强调下，我们做完app很可能会修改，修改时出现删除页面或者改动过大时，响应链接需要重新设置，要不然手机会出现无响应。

            • 我的应用一次性制作不完，可以先保存，等待下一次制作吗？需要修改内容时要怎么做？

              是可以的，您只需要点击制作页面右上方的“确定保存”按钮，即可保存现有制作内容。需要修改内容时打开您所做的应用管理页面，鼠标放到您需要修改的应用上，出现编辑，点击编辑即可进入应用制作页面，找到需要修改的内容修改即可。

            • 我的应用制作完成上架后，每次修改内容都需要生成并且申请上架吗？

              您申请上架并且通过审核的应用，如果需要修改内容都需要在修改内容完成后重新生成并且申请上架，我们需要帮您审核应用内容及时发现您应用出现的问题。

            • 应用修改内容后，安装我应用的人都需要重新下载才可以看到最新内容吗？

              应用制作页面您需要做个应用更新按钮，响应事件选择“应用更新”，安装您应用的用户只需要点击这个按钮就可以更新到最新版本。或者在手机上点下我们公司标志，打开更多页面，点击软件升级就可更新到最新版本。

            • 我不小心把“应用管理”中的应用删掉了，会有什么影响？

              建议最好不要随意删除应用管理中的应用。如果有问题最好直接找客服处理，不要随意删除。

            • 我制作的应用出现生成错误是什么原因？

              应用出现生成错误请检查您制作的应用有没有出现内容删除问题，凡是上传到应用用到的图片素材不可以在素材库中删除，请检查图标、启动页面、内页页面上传图片，解决不了的请联系客服人员，我们会有专门人员帮你解决。

            • 我制作的应用生成不了，排队准备了很久也没有完成生成怎么办？这时候我想修改内容没有编辑按钮该怎么进入？

              应用生成完不成也许是因为您网络太慢，请检查是不是您网络的原因。如果不是网络原因有可能是因为您制作出现错误，或者是您上传图片素材被您删掉，您可以先选择进入应用重新保存生成，如果不是制作问题，重新点击生成应用是可以完成生成的。重新生成还是完不成生成的话，您需要检查下您制作问题。应用在排队等待生成过程中是没有编辑按钮的，这时您想进入应用修改内容需要点击“开始制作”按钮，在页面右侧“已做好的应用”中找到您需要修改的点击即可进入制作页面。

            • 为什么我的手机用微信扫描二维码时，点击“下载”没反应？

              由于微信部分版本对页面访问有限制导致无法下载应用，请在微信中扫描二维码后，点击右上角按钮选择“在浏览器中打开”进行下载。

            • 为什么我的应用制作后，在手机中打开显示启动页后就是白屏或一直停留在启动页面？

              出现这种问题一般为以下几种情况，供您参考：1.链接问题，请查看您的应用的启动页是否链接到了首页，如果没有，则会显示白屏。2.制作问题，请查看您的应用是否有制作除了启动页外的其他页面，如果没有，启动页就是最终页。

            • 为什么我的应用已经修改了首页内容，并已经更新，但是手机中显示的还是原来的内容？

              出现这种问题一般为以下几种情况，供您参考：1.链接问题，请查看您的应用的启动页是否链接到了首页，如果没有，则可能会是这种情况。2.您是否因为网络或者系统原因，没有更新完全，造成更新失败。如果是这种原因，首先要确保您的网络环境良好，安卓机点击手机左键/右键（根据机型），检查更新。iPhone则退出应用，并且双击HOME键，从后台关掉该应用后，重新打开，看是否可以收到更新提示。

            • 为什么我的应用在安卓手机中有些图片不显示，在苹果手机中却显示正常？

              您好，出现这种问题一般为以下几种情况，供您参考：1.图片格式问题，部分用户因为在导图过程中，出现一些格式的损坏，导致手机无法读取图片内容，这时，您只需要重新在photoshop中重新导图即可。2.图片大小问题，安卓应用中的图片超过200KB就可能无法显示，因此请确保您的图片控制在200KB以内。

            • 为什么我的应用在手机中有些页面少背景，或者有些图片有时显示有时不显示？

              您好，出现这种问题一般为以下几种情况，供您参考：1.图片格式问题，部分用户因为在导图过程中，出现一些格式的损坏，导致手机无法读取图片内容，这时，您只需要重新在photoshop中重新导图即可。2.图片大小问题，安卓应用中的图片超过200KB就可能无法显示，因此请确保您的图片控制在200KB以内。

            • 我在制作页面添加了文本控件，制作页面上显示正常，可是我的手机上里面的文字却显示不全？

              您好，出现这种问题一般为以下几种情况，供您参考：1.图片格式问题，部分用户因为在导图过程中，出现一些格式的损坏，导致手机无法读取图片内容，这时，您只需要重新在photoshop中重新导图即可。2.图片大小问题，安卓应用中的图片超过200KB就可能无法显示，因此请确保您的图片控制在200KB以内。
